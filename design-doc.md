# 仙踪奇缘 · 游戏设计文档 (GDD)

> 一款受《梦幻西游》启发的 2D 竖屏回合制 RPG · 移动端 H5 / 微信小游戏方向
> 版本: v0.1 (Demo 阶段)
> 文档维护: ice
> 最后更新: 2026-05-07

---

## 1. 项目概述

### 1.1 一句话愿景

**"在掌中重温童年的回合制 MMO 江湖"** —— 用现代轻量 H5 技术,把端游级的回合制策略乐趣装进竖屏手机里。

### 1.2 核心立项要素

| 维度       | 设定                                                 |
|------------|------------------------------------------------------|
| 类型       | 2D 俯视角 · 回合制 · 角色扮演 (RPG)                  |
| 视角       | 俯视 45° (类似梦幻西游 / 口袋妖怪)                    |
| 平台       | H5 (浏览器) → 微信小游戏 / iOS / Android (Cordova)   |
| 屏幕方向   | 竖屏为主, 战斗界面也按竖屏布局                       |
| 美术风格   | 中国风水墨像素 (Demo 阶段先用矢量纯色, 后期切像素图) |
| 目标受众   | 25-40 岁 怀旧向手游玩家, 通勤/碎片时间用户            |
| 商业模式   | 免费 + 内购 (装备/坐骑/外观), 无强制对战              |
| 引擎       | 原生 Canvas 2D (Demo) → Cocos Creator / Phaser (正式) |

### 1.3 与"梦幻西游"的差异化

不是 1:1 复刻, 而是抓三件事:

1. **回合制的策略深度** —— 出手速度、五行相克、技能联动、宠物 (召唤兽) 体系
2. **轻松的社交氛围** —— 帮派、组队、师徒,但都做成轻量异步 (因为是手游)
3. **熟悉的世界框架** —— 三界 (人/魔/仙)、门派、坐骑、家园等系统骨架借鉴

减法的部分:取消硬核 PVP/PK 和封测年代的复杂数值, 让新手 5 分钟入门。

## 2. 玩法循环

### 2.1 核心循环 (短线, 一次会话 5-10 分钟)

```
进入场景 → 接日常任务 → 探索/触怪 → 回合制战斗 → 获得经验/金币/掉落
   ↑                                                               ↓
   └─────────────── 升级 / 学技能 / 升装备 ←──────────────────────┘
```

### 2.2 中线循环 (一周左右)

- 主线剧情推进 → 解锁新场景 / 新地图
- 门派任务 → 解锁新技能、新坐骑
- 召唤兽培养 → 抓捕 → 培养 → 进阶 → 合成

### 2.3 长线目标 (1-3 个月)

- 通关主线 (60 关左右)
- 帮派建设 / 副本周常 / 师门排名
- 限时活动 (节日/双倍/抽奖)

## 3. 系统设计

### 3.1 角色系统

#### 3.1.1 五大门派 (各 2 系, 共 10 个职业大类)

| 门派     | 阵营 | 职业风格              | 核心机制              |
|----------|------|-----------------------|-----------------------|
| 蜀山派   | 仙   | 物理/法术双修         | 高爆发, 中防御         |
| 普陀山   | 仙   | 治疗/辅助             | 全队回血、复活        |
| 化生寺   | 仙   | 群体辅助/控制         | 加防、清状态          |
| 方寸山   | 仙   | 法术控制              | 定身、封印            |
| 天宫     | 仙   | 雷系输出 / Tank       | 群法 + 高生命         |
| 大唐官府 | 人   | 物理高输出            | 单体爆击, 偷袭        |
| 女儿村   | 人   | 群体物理 + 减疗       | 命中、毒系             |
| 龙宫     | 魔   | 群体法术              | 高魔法 / 低生命        |
| 魔王寨   | 魔   | 物理坦克              | 高血、嘲讽             |
| 阴曹地府 | 魔   | 控制 / 法术 DOT       | 鬼魂、亡灵召唤         |

> Demo 阶段只需要做一个**通用职业 (剑侠)**, 后续按门派拆分。

#### 3.1.2 角色属性

每个角色有以下基础属性:

| 属性 | 含义                | 影响                         |
|------|---------------------|------------------------------|
| HP   | 生命值              | 归零进入濒死/复活流程         |
| MP   | 魔法值              | 释放技能消耗                  |
| ATK  | 物理攻击            | 影响普攻和物理技能伤害         |
| MAG  | 法术攻击 (后续加入) | 影响法术技能伤害               |
| DEF  | 物理防御            | 减伤                          |
| RES  | 法术抗性 (后续加入) | 法术减伤                      |
| SPD  | 速度                | 决定回合内出手顺序             |
| CRT  | 暴击率 (后续加入)   | 暴击概率, 暴击伤害 1.5×       |
| LCK  | 幸运 (后续加入)     | 闪避、抓宠成功率              |

属性来源: **基础值 + 等级加成 + 装备 + 称号 + Buff**

#### 3.1.3 五行系统

仿梦幻西游, 使用 **金/木/水/火/土** 五行相克:

```
金克木 → 木克土 → 土克水 → 水克火 → 火克金 → (循环)
```

战斗中, 五行克制的攻击伤害 +25%, 被克制时伤害 -25%。每个职业、每件装备、每只召唤兽都有五行属性。

### 3.2 战斗系统

#### 3.2.1 回合流程

```
回合开始
  ├─ 全员根据 SPD 排序 (含双方 + 召唤兽)
  ├─ 玩家依次操作 (限时 30s, 否则默认普攻)
  │    可选动作: 攻击 / 技能 / 防御 / 物品 / 逃跑 / 捕捉
  ├─ 全员按队列出手, 计算伤害/治疗/状态
  ├─ Buff/Debuff 持续时间 -1
  └─ 检查胜负 → 进入下一回合或结算
```

#### 3.2.2 伤害公式

```
基础伤害 = (攻击者.ATK × 技能倍率) - 被攻击者.DEF × 0.6
五行修正 = ×1.25 (克制) / ×0.75 (被克制) / ×1.0 (无关)
随机浮动 = ×(0.95~1.05)
防御中    = ×0.5
最终伤害 = max(1, round(基础 × 五行 × 随机 × 防御))
```

> Demo 版只实现了 **基础伤害 + 防御中减半 + 随机浮动**, 五行相克在正式版引入。

#### 3.2.3 状态效果 (Buff/Debuff)

| 状态     | 效果                              | 持续         |
|----------|-----------------------------------|--------------|
| 中毒     | 每回合扣 5% 最大 HP                | 3 回合        |
| 灼烧     | 每回合扣固定值伤害                 | 2 回合        |
| 封印     | 不能使用技能                      | 2 回合        |
| 麻痹     | 50% 无法行动                      | 2 回合        |
| 防御提升 | DEF +25%                          | 直到本场结束 |
| 加速     | SPD +30%                          | 3 回合        |

#### 3.2.4 战斗类型

- **野战** (随机遇敌): 单次, 经验金币, 可逃跑
- **任务战** (剧情): 不可逃跑, 通关给道具/解锁
- **副本战** (多波): 3-5 波连续, 中间不能补给, 通关大额奖励
- **PVP 切磋**: 不掉装备, 排行榜
- **Boss 战**: 长血条, 多阶段, 阶段切换有 CG

### 3.3 召唤兽 (宠物) 系统

> 梦幻西游的灵魂玩法, 必须做。

#### 3.3.1 抓捕

野战中, 玩家可在敌方残血时使用"困兽诀"。成功率取决于:

```
成功率 = base × (1 - hp_ratio) × (1 + LCK × 0.01) × (难度修正)
```

#### 3.3.2 培养

| 维度        | 说明                                |
|-------------|-------------------------------------|
| 资质        | 出生时随机, 决定成长上限             |
| 成长率      | 每级属性增长, 看资质                |
| 技能槽      | 一开始 2-3 个, 可通过打书扩充       |
| 等级        | 与玩家共战获得经验                  |
| 灵性        | 进阶相关 (后期)                     |
| 五行        | 一只宠一种五行                      |

#### 3.3.3 合成 / 进阶

合宠 = 选两只为父母, 子代继承父母最优值并随机加成。这是梦幻西游的"洗宠"经济基石,也是付费点。

#### 3.3.4 战斗中的召唤兽

每个玩家可上场 1 只主战召唤兽 (后期可上场 2 只)。它单独占用一个回合位, 有独立 HP/MP/SPD。

### 3.4 任务系统

| 任务类型    | 数量 / 周期 | 奖励           | 备注                       |
|-------------|-------------|----------------|----------------------------|
| 主线        | ~60 关      | 大额经验/解锁   | 推动剧情, 不可重复          |
| 师门日常    | 10/天        | 师门贡献/经验   | 日常打怪/送信              |
| 抓鬼        | 20/天        | 经验/金钱      | 经典玩法, 时限性            |
| 押镖        | 1/天         | 大额金币        | 路上随机遇袭                |
| 帮派任务    | 3/天         | 帮派贡献        | 团队合作                    |
| 节日/限时   | 不定         | 限定外观/坐骑   | 运营驱动                    |

### 3.5 经济系统

#### 3.5.1 货币

- **金币**: 主货币, 战斗 / 任务获得, 用于装备打造、修理
- **银两**: 副货币, 任务获得, 用于召唤兽进阶、特殊兑换 (防止外挂)
- **元宝**: 充值货币 (1 元 = 10 元宝), 可买外观、加速、抽卡

#### 3.5.2 装备

5 个部位 (头/衣/腰/手/鞋) + 武器 + 项链 + 戒指 + 称号 = **9 件**

每件装备有 4-7 条属性, 可通过"打造"获得不同品质 (白/绿/蓝/紫/橙/红)。

#### 3.5.3 商城 / 摆摊

玩家间可摆摊买卖装备/宠物/材料。摆摊收 5% 手续费 (回收金币)。

### 3.6 社交系统

| 系统     | 描述                                                      |
|----------|-----------------------------------------------------------|
| 师徒     | 高等级带低等级, 双方都获得加成                              |
| 结拜     | 4 人小团, 共享资源, 称号同步                                |
| 帮派     | 50-100 人公会, 帮派副本/任务/排名                          |
| 好友     | 加好友、聊天、组队                                          |
| 队伍     | 3 人队, 一起打副本/任务                                    |
| 离线挂机 | 委托队友帮挂, 收益分成 (手游特有, 适配碎片时间)            |

## 4. 关卡 / 内容设计

### 4.1 世界观

**三界初分, 妖魔渐起**。

千年前妖王降世, 三界共同封印于"幽暗洞穴"深处。如今封印松动,各路修行人各自下山, 寻找碎片、调查异象。玩家是其中一位刚下山的少年/少女, 先回村 (新手村), 再入森林历练, 最终探入洞穴对抗复苏的妖王。

### 4.2 Demo 三幕场景 (本项目当前实现)

#### 场景 1: 桃源村 (新手区)

- **气氛**: 阳光、花、几栋小屋、井
- **NPC**:
  - **李长老** (剧情触发) — 给予出发动机
  - **王掌柜** (商人占位) — 后续接通商店系统
  - **小丫** (彩蛋) — 教学型对话
- **怪物**: 无 (安全区)
- **传送出口**: 南门 → 迷雾森林

#### 场景 2: 迷雾森林 (战斗区)

- **气氛**: 深绿、雾气、池塘
- **NPC**:
  - **云游道士** (训练战) — 对话结束后切磋
- **怪物**:
  - **青史莱姆** ×2 (Lv1, 经验 8)
  - **林地哥布林** ×1 (Lv2, 经验 15, 会用"猛击")
- **传送**: 北 → 桃源村, 东南 → 幽暗洞穴

#### 场景 3: 幽暗洞穴 (BOSS 区)

- **气氛**: 阴暗、回声、磷光
- **NPC**:
  - **神秘老者** — 一次性免费回血点
- **怪物**:
  - **幽冥蝙蝠** ×2 (高速, 低血)
  - **洞穴守护者** (BOSS) — 80 HP, 会用"怒吼斩"+"岩石护盾", 不可逃跑
- **传送**: 北 → 迷雾森林

## 6. 技术架构

### 6.1 Demo 阶段技术栈

```
HTML5 / CSS3
   └── Canvas 2D Context
         └── 自研轻量游戏框架
              ├── 状态机 (explore / dialogue / battle / transition)
              ├── 输入抽象 (键盘 + 触摸虚拟摇杆 + 动作键)
              ├── 场景图 (瓦片地图 + NPC + 怪物)
              ├── 实体排序绘制 (按 y 排序解决遮挡)
              └── UI 渲染 (HUD / 对话框 / 战斗菜单)
```

### 6.2 正式版技术选型 (建议)

| 模块         | 选型                              | 理由                                |
|--------------|-----------------------------------|-------------------------------------|
| 引擎         | **Cocos Creator 3.x**             | 国内文档丰富, 社区活跃, 微信小游戏适配好 |
| 网络         | WebSocket (协议 Protobuf)          | 实时战斗 / 聊天                       |
| 后端语言     | Go (战斗服) + Node.js (业务服)    | 战斗服强一致, 业务服快速迭代           |
| 数据库       | MySQL (账号/订单) + Redis (会话)  | 标配                                  |
| 实时同步     | 帧同步 (战斗) / 状态同步 (世界)    | 回合制不需要高频帧同步               |

| 资源管理     | CDN + 热更                          | 美术资源大, 必须热更                |

## 8. 商业化设计

### 8.1 付费深度

- **轻度玩家 (0~50 元/月)**: 月卡 + 解锁外观
- **中度玩家 (50~500 元/月)**: 召唤兽进阶 + 装备宝石
- **大 R (500+ 元/月)**: 限定坐骑 / 称号 / 跨服比武

### 8.2 防止 P2W 红线

- **PVE 不卖战力** (装备只能打造, 不能直接买)
- **PVP 同区段同战力**, 跨区匹配只有切磋, 无奖励
- **召唤兽** 可付费抽 (但顶级只能合成, 给非氪金玩家上限)

### 8.3 关键转化点

1. 新手关 7 天内引导一次 6 元 月卡
2. 等级 30 后引导第一次 30 元 充值 (送限定外观)
3. 帮派加入后引导 帮派福利包
